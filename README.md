# Sudoku Solver

原生 Python 数独求解器，带图形界面。

## 启动方式

```bash
# 激活 conda 环境
conda activate sudoku

# 运行程序
python sudoku_solver.py
```

## 功能

| 按钮 | 说明 |
|------|------|
| 求解 | 回溯算法自动求解 |
| 提示 | 填入一个空格的值（绿色高亮） |
| 检错 | 标出填写错误的单元格（红色） |
| 保存题目 | 将蓝色数字保存为 .txt 文件 |
| 加载题目 | 从 .txt 文件加载题目 |
| 清空 | 重置棋盘 |

### 颜色含义

| 颜色 | 含义 |
|------|------|
| 浅蓝 | 原始题目数字 |
| 浅绿 | 求解后填入的答案数字 |
| 白色 | 空格子 |

### 操作方式

- 直接点击格子输入数字（1-9），Delete 清空
- 方向键移动焦点
- 也可使用底部数字键盘

## 文件格式

.txt 保存格式为 9 行，每行 9 个字符（数字或 `.` 表示空格），例如：

```
5..7..9..
..8.....3
....4.2..
....1.6..
.7.....5.
..3.9....
..5.....1
9.......7
..2..6...
```
